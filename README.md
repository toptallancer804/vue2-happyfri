# happyfri

一个简单的vue2入门项目，熟悉如何利用vue + vue-router + vuex 在初期搭建一个项目

## 项目运行
``` bash
# 安装依赖
npm install

# 开启本地服务器localhost:8080
npm run dev

# 发布环境
npm run build
```
